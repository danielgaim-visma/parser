# DocxParserGUI

## Installation Instructions

### macOS

1. Download the `DocxParserGUI.zip` file.
2. Open Terminal and navigate to the directory containing the downloaded zip file.
3. Run the following commands:

   ```
   unzip DocxParserGUI.zip
   chmod +x install.sh
   ./install.sh
   ```

4. The application will be installed in your Applications folder.

### Running the Application

Open the Applications folder and double-click on DocxParserGUI to run the application.
